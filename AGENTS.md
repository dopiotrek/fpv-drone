# Project Context: FPV Drone Build

## Constraints
- **Budget:** ~200 EUR (Hard limit for drone components).

- **Tools:** Limited (No 3D printer).

- **Frame:** TBS Source One V5 (5-inch).
- **Control Link:** ELRS 2.4GHz.
- **Video:** Analog.

## User Preferences

- Phase-based project structure.
